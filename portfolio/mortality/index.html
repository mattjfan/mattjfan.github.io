Mortality

Web Development | Solo | 2017

Mortality is a small Motivation clone that I built for Firefox to read out my age in "real-time".

Given the limitations of Firefox's web extensions API, Mortality is currently an HTML page hosted on github pages, which I'm using as my Firefox homepage, although I intend to port it to an extension if Firefox ever updates the API. In addition to the basic counting up feature, I also added a "Mortality" counter, which gives an estimated "percent of the way through", based on publicly available estimated lifespans for men in the US.

You can check out the original version here, a customizable version (still a WIP) here, and the source code on Github here.
